# Calendar.ai

## Version 0.001

This is the initial release of the Calendar.ai application.

### Features
- **Responsive Design:** Full-screen layout that adapts to different screen sizes.
- **Hierarchical Views:**
    - **Decade View:** Select years from a grid.
    - **Year View:** Select months.
    - **Month View:** View days of the month.
    - **Day View:** View hourly timeline.
- **Navigation:** Seamless transition between views using headers and back buttons.
- **Modern UI:** Clean aesthetic with Google Fonts (Outfit) and Material Symbols.

### How to Run
Simply open `index.html` in any modern web browser.

### Tech Stack
- HTML5
- CSS3 (Vanilla)
- JavaScript (Vanilla)
